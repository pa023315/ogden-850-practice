# English 5000 Practice

英文 5000 詞練習網站，使用 ECDICT 前 5000 常用詞建立練習資料。

## 功能

- 每日練習自動排題
- 四選一選中文翻譯
- 四選一選英文翻譯
- 四選一單字填空
- 英檢風格四選一補空題
- 多益風格四選一補空題
- 單字朗讀、句子朗讀、串念單詞、串念句子
- 錯誤紀錄
- 瀏覽器本機儲存學習進度
- 以簡化 SM-2 間隔重複邏輯排複習題

英檢與多益題型為自動生成的仿真練習題，不含官方考古題內容。

## 詞庫來源

- ECDICT: Free English to Chinese Dictionary Database, MIT License
- Repo: https://github.com/skywind3000/ECDICT

## 使用方式

直接開啟 `index.html`，或用任何靜態網站服務部署本資料夾。
